RAG System - Chat với Tài Liệu PDF

Upload a PDF file and ask questions — the AI will answer based on the document content! The system uses Retrieval-Augmented Generation (RAG) with the Qwen2.5:7B LLM running locally via Ollama, combined with a FAISS vector database for contextual retrieval and LangChain to generate intelligent responses. It features a modern Streamlit interface with automatic Vietnamese–English bilingual support.

Key Features & Architecture:

Document Processing: PDFPlumber extracts text, chunks into 1000-character segments with 100-char overlap, generates 768-dimensional embeddings using MPNet multilingual model, stores in FAISS for similarity search (retrieves top-3 relevant chunks per query).

Query Pipeline: Automatic Vietnamese/English detection → Custom prompt engineering → Vector retrieval → Qwen2.5:7b (7B parameters) generates contextual answers → No API costs, complete privacy with local LLM.

Technologies & Requirements:

Python 3.8+ Streamlit LangChain 0.3.16 FAISS Ollama Qwen2.5:7b HuggingFace PDFPlumber Sentence-Transformers

System Requirements: Python 3.8+ | RAM 8GB+ (16GB recommended) | ~10GB storage for Qwen2.5:7b model | Ollama installed

Technical Component Details:

Python 3.8+ & Streamlit

Python là ngôn ngữ lập trình core cho ML/AI với ecosystem phong phú (NumPy, Pandas). Streamlit 1.41.1 cung cấp web framework để build interactive UI chỉ với Python thuần - không cần HTML/CSS/JS. Session state management cho chat history, file upload widget cho PDF, và real-time streaming response từ LLM.

LangChain 0.3.16

Framework orchestration cho LLM applications. Sử dụng OllamaLLM wrapper để tích hợp Qwen2.5, RecursiveCharacterTextSplitter cho intelligent chunking (tôn trọng paragraph/sentence boundaries), RetrievalQA chain kết hợp retriever + LLM, và PromptTemplate cho context injection + instruction engineering.

FAISS (Facebook AI Similarity Search)

Vector database tối ưu cho similarity search với IndexFlatL2 (exact L2 distance). Lưu trữ 768-dim embeddings, search algorithm trả về top-k most similar vectors trong O(n) time. Serialization support cho persistent storage (save/load vector index). Xử lý được millions of vectors efficiently.

Ollama + Qwen2.5:7b

Ollama là local LLM runtime (như Docker cho AI models) - download, run, manage models dễ dàng. Qwen2.5:7b (Alibaba Cloud) có 7 billion parameters, hỗ trợ multilingual (Vi/En), context window 32K tokens, instruction-tuned for chat/QA, và chạy hoàn toàn offline - zero API costs, complete data privacy.

HuggingFace + Sentence-Transformers

HuggingFace cung cấp model hub với hàng nghìn pretrained models. Sentence-Transformers library tạo semantic embeddings - sử dụng paraphrase-multilingual-mpnet-base-v2 model (50+ languages, 768 dimensions) để convert text thành dense vectors. Cosine similarity giữa vectors đo semantic meaning tương đương.

PDFPlumber

Python library extract text từ PDF files với high accuracy. Parse PDF structure (text, tables, metadata), xử lý complex layouts, preserve formatting, và handle multi-page documents. Alternative cho PyPDF2/pdfminer với API đơn giản hơn và better text extraction quality.

Why This Tech Stack?

100% Local & Private: Ollama + Qwen2.5 = zero cloud dependency, không gửi data ra external APIs (GDPR compliant). Production-Ready: FAISS scaling đến millions vectors, LangChain có robust error handling. Multilingual: MPNet embeddings + Qwen2.5 support Vietnamese và English natively. Cost-Effective: One-time setup, zero API fees, chạy unlimited queries.
